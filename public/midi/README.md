# MIDI Song Slots

Place MIDI files here to make the app's built-in song buttons work:

- `mrs-magic-strings-version.mid` - generated from the Mrs Magic YouTube video extraction workflow
- `stress-relief-late-night-drive-home.mid` - beginner piano-coach arrangement generated by `npm run generate:stress-relief`

The app also supports loading any `.mid` or `.midi` file manually, or pasting a direct MIDI URL into the Song panel.
